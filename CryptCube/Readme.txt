===============================================================================
    密码学盒子 CryptCube 项目概述
===============================================================================

密码学盒子是一个密码学辅助学习工具。它提供了全面的、经典的、基础的密码算法中间演示过程，和一些采用密码编码技术实现的小工具。

直观地向您展示了各类密码的原理与应用。因此非常适合高校信息安全相关专业密码学课程的辅助教学工作。

同时也可作为非信息安全从业人员探索密码学奥秘的一个快速途径。

更重要的是，它免费，完全免费，并提供源程序。帮您在深入了解密码机制的同时，无任何限制地学习密码算法的C语言描述并迅速展开编程实践。

这个软件使用VS2008+SP1环境里的MFCRibbon功能包构建。

源程序中，对各个类、方法、对象、变量、资源文件都按照了统一的命名规则进行命名，

并竭尽所能地做出了详尽的注释，方便童鞋们阅读。




应用程序向导已为您创建此 CryptCube 应用程序。此应用程序不仅演示使用 Microsoft 基础类的基本知识，而且可作为编写应用程序的起点。

此文件包含组成 CryptCube 应用程序的各个文件的内容摘要。

CryptCube.vcproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件。
    它包含有关生成文件的 Visual C++ 版本的信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

CryptCube.h
    这是应用程序的主要头文件。它包括其他项目特定的头文件(包括 Resource.h)，并声明 CCryptCubeApp 应用程序类。

CryptCube.cpp
    这是包含应用程序类 CCryptCubeApp 的主要应用程序源文件。

CryptCube.rc
    这是程序使用的所有 Microsoft Windows 资源的列表。它包括 RES 子目录中存储的图标、位图和光标。此文件可以直接在 Microsoft Visual C++ 中进行编辑。项目资源位于 2052 中。

res\CryptCube.ico
    这是用作应用程序图标的图标文件。此图标包括在主要资源文件 CryptCube.rc 中。

res\CryptCube.rc2
    此文件包含不是由 Microsoft Visual C++ 编辑的资源。您应该将不可由资源编辑器编辑的所有资源放在此文件中。

/////////////////////////////////////////////////////////////////////////////

对于主框架窗口:
    项目包含标准 MFC 界面。

MainFrm.h, MainFrm.cpp
    这些文件包含框架类 CMainFrame，该类派生自
    CFrameWnd 并控制所有 SDI 框架功能。

res\Toolbar.bmp
    此位图文件用于为工具栏创建平铺图像。
    初始工具栏和状态栏是在 CMainFrame 类中构造的。使用资源编辑器编辑此工具栏位图，并更新 CryptCube.rc 中的 IDR_MAINFRAME TOOLBAR 数组以添加工具栏按钮。
/////////////////////////////////////////////////////////////////////////////

应用程序向导创建一个文档类型和一个视图:

CryptCubeDoc.h，CryptCubeDoc.cpp - 文档
    这些文件包含 CCryptCubeDoc 类。编辑这些文件可以添加特殊文档数据并可实现文件保存和加载(通过 CCryptCubeDoc::Serialize)。

CryptCubeForm.h，CryptCubeForm.cpp - 文档的视图
    这些文件包含 CCryptCubeForm 类。
    CCryptCubeForm 对象用于查看 CCryptCubeDoc 对象。

ActiveX 控件

其他标准文件:

StdAfx.h, StdAfx.cpp
    这些文件用于生成名为 CryptCube.pch 的预编译头(PCH)文件和名为 StdAfx.obj 的预编译类型文件。

Resource.h
    这是标准头文件，它定义新资源 ID。
    Microsoft Visual C++ 将读取并更新此文件。
